# 🎵 夸克音乐搜索助手

多平台音乐搜索与夸克网盘链接获取工具，支持单首搜索、批量导出、排行榜推荐

[English](README_EN.md) | 简体中文

## ✨ 功能特性

### 核心功能
- 🔍 **单首搜索** - 横向双栏界面，左栏搜索，右栏显示结果列表
- 🎵 **音乐推荐** - 多平台热门榜单推荐（QQ音乐、酷狗、网易云、Apple Music）
- 📦 **批量导出** - 导入歌单文件，一键批量获取夸克网盘链接
- 🏆 **原唱识别** - 从豆瓣音乐获取原唱信息，自动标注原唱版本
- ⭐ **收藏功能** - 收藏喜欢的歌曲，自动持久化保存，重启不丢失
- 💾 **多种保存** - 单首保存、批量保存、收藏导出等多种方式
- 🎶 **音乐播放器** - 内置Apple Music风格播放器，支持播放控制

### 多平台排行榜

## 🚀 快速开始

### 环境要求

- Python 3.8+
- Windows / macOS / Linux

### 安装依赖

```bash
pip install -r requirements.txt
```

### 启动桌面版

双击运行或命令行启动：

```bash
python desktop.py
```

桌面版会在窗口中运行，无需浏览器。

## 📖 使用说明

### 单首搜索

1. 在底部导航栏点击「🔍 搜索」
2. 在左侧输入歌曲名（歌手可选）
3. 选择搜索源（歌曲宝/XM无损音乐/动听音乐）
4. 点击「🔍 搜索」按钮
5. 右侧显示搜索结果列表，标注原唱和匹配度
6. 点击歌曲卡片展开详情，查看夸克链接
7. 可进行：播放、复制链接、打开链接、收藏

### 批量导出

1. 在底部导航栏点击「📦 批量」
2. 第一步：粘贴歌单内容（每行一首，格式：歌曲名 - 歌手）
3. 第二步：点击「开始导出」进行批量搜索
4. 第三步：查看结果，支持导出成功链接

### 收藏管理

1. 搜索到喜欢的歌曲后点击「⭐ 收藏」
2. 在底部导航栏点击「⭐ 收藏」查看收藏列表
3. 支持：导出收藏、清空收藏

### 音乐推荐

1. 在底部导航栏点击「🎵 推荐」
2. 在左侧选择平台（QQ音乐/酷狗/网易云/Apple Music）
3. 在左侧选择榜单
4. 右侧显示榜单歌曲列表，点击「🔍 搜索」一键搜索该歌曲
5. 数据自动缓存到本地，6小时自动刷新一次
6. 可点击「🔄 刷新」按钮手动刷新

### 音乐播放

1. 在搜索结果中点击歌曲的播放按钮
2. 底部播放器会显示当前播放歌曲信息
3. 支持：播放/暂停、上一首、下一首、音量调节
4. **收纳播放器**：点击播放器左上角的收纳按钮，播放器会隐藏到底部导航栏
5. **展开播放器**：点击底部导航栏中间的音乐封面图标即可重新展开
6. 播放时封面会旋转，显示动态渐变效果

## 📝 歌单文件格式

支持以下格式（每行一首）：

**格式一（推荐）：**
```
1. 晴天 - 周杰伦
2. 稻香 - 周杰伦
3. Good Time - Owl City
```

**格式二：**
```
晴天 - 周杰伦
稻香 - 周杰伦
```

**格式三：**
```
晴天
周杰伦
稻香
周杰伦
```

> 💡 支持汽水音乐导出的歌单格式，空行和注释行自动跳过

## ❓ 常见问题

### Q: 搜索不到歌曲怎么办？
A: 可以尝试：简化歌曲名、只填歌名不填歌手、确认歌曲名拼写是否正确。

### Q: 原唱信息准确吗？
A: 原唱信息从豆瓣音乐获取，大部分流行歌曲准确率较高。如果识别有误，以实际为准。

### Q: 批量搜索中断了怎么办？
A: 命令行版本支持断点续传，重新运行会自动跳过已处理的歌曲。

### Q: 夸克链接能直接下载吗？
A: 需要将链接保存到自己的夸克网盘后才能下载。复制链接到夸克网盘网页版，会自动弹出保存提示。

### Q: 排行榜数据多久更新一次？
A: 数据自动缓存到本地，6小时自动刷新一次。也可以点击「🔄 刷新」按钮手动刷新。

### Q: 哪些平台的排行榜可用？
A: 当前支持QQ音乐、酷狗音乐、网易云音乐、Apple Music四个平台的排行榜。
